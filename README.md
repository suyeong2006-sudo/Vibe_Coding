# Vibe_Coding
개요: HTML·CSS·JS 기반의 15장짜리 풀스크린 웹 슬라이드쇼입니다. 설치 없이 브라우저로 실행되며 다크 테마의 SF 디자인이 특징입니다.  조작 및 UI: 방향키(이동), Tab(개요 모드), F(전체화면) 단축키와 마우스로 제어합니다. 진행 바와 힌트 바, 대형 인덱스로 직관성을 높였습니다.  효과: 터미널 타이핑 효과와 슬라이딩 전환 애니메이션이 적용되었습니다.  내용: 자연어 기반 개발인 ‘바이브 코딩’의 개념과 워크플로우, 실리콘밸리 트렌드를 다룹니다. 또한 기술 부채, 제어 불가능한 부작용 등 생산 단계의 한계를 짚어보고 미래 엔지니어에게 필요한 메타 인지 역량을 제시합니다.
